# Japan Housing Troubleshooter

Free interactive tool to check if your move-out charges are fair, based on official MLIT guidelines. Built for foreigners living in Japan.

## Features

- Move-out cost checker with per-item verdicts
- Depreciation calculator based on MLIT guidelines
- Bilingual dispute email templates (EN/JP)
- Escalation flowchart (self-negotiate > Consumer Affairs > Houterasu > lawyer)
- Move-in photo checklist
- Mobile-first, no signup required
